CSS Introduction

What is CSS?

CSS (Cascading Style Sheets) is used to style and design web pages by controlling how elements look, such as colors, fonts, and layout. It works with HTML to make websites more attractive and user-friendly. Without CSS, web pages would appear plain and unstyled.

Types to add css to html

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Internal CSS

CSS is written directly inside an HTML tag using the style attribute.

Internal CSS

CSS is written inside a style tag in the head section of the HTML file.

External CSS

CSS is written in a separate .css file, and linked to the HTML using a link tag.
